# Heartbeat Checklist

When the system sends a heartbeat, review this checklist. If everything is clear, respond with only `[HEARTBEAT_OK]`.

## Quick Checks
1. **Assigned tasks** — Review your current assignments. Are you blocked on anything?
2. **Pending deliverables** — Check if you have any completed work that needs to be reported to your manager.
3. **Blockers** — If you are stuck or need resources/credentials, escalate to your manager.

## Rules
- Do NOT narrate. Just check and report.
- If all checks pass: respond with exactly `[HEARTBEAT_OK]`
- If something needs action: handle it and briefly summarize.
