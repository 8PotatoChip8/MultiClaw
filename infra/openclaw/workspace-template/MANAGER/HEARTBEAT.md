# Heartbeat Checklist

When the system sends a heartbeat, review this checklist. If everything is clear, respond with only `[HEARTBEAT_OK]`.

## Quick Checks
1. **Worker status** — Check if your workers are active and making progress on their assigned tasks.
2. **Pending requests** — Check for any pending requests from your workers that need your approval.
3. **Task progress** — Review any in-progress tasks and note blockers or delays.
4. **CEO updates** — If there are significant updates to report, DM your CEO with a brief status.

## Rules
- Do NOT narrate. Just check and report.
- If all checks pass: respond with exactly `[HEARTBEAT_OK]`
- If something needs action: handle it and briefly summarize.
